# Análise de Empresas Unicórnio - 2021

Este projeto tem como objetivo praticar e aprofundar o uso da biblioteca Pandas por meio da análise de um Dataset de startups que atingiram o status de unicórnio em 2021.
> São consideradas unicórnios as empresas privadas avaliadas em US$ 1 bilhão ou mais, conceito popularizado por Aileen Lee em 2013 para destacar a raridade e o rápido crescimento desse tipo de negócio.

A proposta foi explorar o conjunto de dados, compreender suas características e extrair alguns insights interessantes sobre o cenário de unicórnios ao longo do ano de 2021.

O dataset utilizado para as análises pode ser encontrado na pasta raw na fonte do projeto ou pelo link: https://www.kaggle.com/datasets/prasertk/unicorn-decacorn-hectocron-in-2021.
